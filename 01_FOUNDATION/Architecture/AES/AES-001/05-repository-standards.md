# AES-001 — Repository Standards

---

# Purpose

Defines the standard structure for all AURA repositories.

---

# Standard Layout

```
project/

docs/

src/

tests/

scripts/

configs/

assets/

tools/

.github/

README.md

LICENSE
```

---

# Repository Rules

- One Responsibility Per Repository
- README Required
- License Required
- CHANGELOG Required
- Version Tags Required

---

# Branch Strategy

- main
- develop
- feature/*
- hotfix/*
- release/*

---

# End of Repository Standards
