# AES-001 — Documentation Standards

---

# Purpose

Documentation is treated as a first-class engineering artifact.

---

# Documentation Requirements

Every project shall contain:

- README
- Installation Guide
- User Guide
- API Documentation
- Architecture Documentation
- Changelog

---

# Writing Principles

- Clear
- Accurate
- Versioned
- Searchable
- Consistent

---

# Documentation Formats

- Markdown
- Mermaid
- OpenAPI
- UML
- Architecture Decision Records

---

# End of Documentation Standards
