# AES-001 — Coding Standards

---

# Purpose

This chapter defines coding conventions used across all AURA projects.

---

# General Standards

- Follow SOLID Principles
- DRY (Don't Repeat Yourself)
- KISS (Keep It Simple)
- YAGNI (You Aren't Gonna Need It)

---

# Naming Conventions

## Classes

PascalCase

Example:

UserManager

---

## Variables

camelCase

Example:

userProfile

---

## Constants

UPPER_SNAKE_CASE

Example:

MAX_RETRY_COUNT

---

## Files

kebab-case

Example:

user-service.py

---

# Code Requirements

- Meaningful Names
- Small Functions
- Single Responsibility
- Error Handling
- Logging
- Comments Only When Necessary
