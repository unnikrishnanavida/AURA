# Appendix A — Glossary

| Term | Definition |
|------|------------|
| Knowledge Graph | A semantic network of entities and relationships |
| Ontology | Formal representation of concepts and relationships |
| Embedding | Numerical representation of data for semantic similarity |
| Retrieval-Augmented Generation (RAG) | AI technique combining retrieval with generation |
| Episodic Memory | Memory of events and experiences |

# Appendix C — Knowledge Principles

- Knowledge is contextual.
- Knowledge is versioned.
- Knowledge is explainable.
- Knowledge evolves continuously.
- Knowledge is reusable.
- Knowledge is secure by design.
