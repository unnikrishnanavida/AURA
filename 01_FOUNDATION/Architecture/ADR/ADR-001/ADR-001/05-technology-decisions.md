# ADR-001 — Technology Decisions

---

# Purpose

This chapter records technology selection decisions for the AURA platform.

---

# Decision 001

Technology Stack

Decision:

Python shall be the primary backend programming language.

Status:

Approved

---

# Decision 002

Frontend

Decision:

React shall be used for modern web applications.

Status:

Approved

---

# Decision 003

Database

Decision:

PostgreSQL shall be the primary relational database.

Status:

Approved

---

# Decision 004

Containerization

Decision:

Docker shall be used for application packaging.

Status:

Approved

---

# End of Technology Decisions
